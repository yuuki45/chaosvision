# 📱 アプリ要件定義書：中二スキャナー - CHAOS VISION -

## 1. アプリ概要

**名称（仮）**: 中二スキャナー - CHAOS VISION -  
**ジャンル**: エンタメ / カメラ AR / 中二病診断  
**対象 OS**: iOS / Android（クロスプラットフォーム：Flutter 推奨）  
**コンセプト**:  
現実世界に存在するあらゆる物体に"異名"と"裏設定"を付与し、中二病的視点で再解釈する AR スキャンアプリ。  
ユーザーはスマホをかざすだけで、この世界の"真の姿"を見ることができる。

---

## 2. 想定ユーザー

- 中二病が好きな人（ネタ・自己投影問わず）
- 日常に遊び心を求める若者・大人
- SNS でネタ投稿したい人（TikTok/X 向け）
- 学生やクリエイター（創作のネタとして）

---

## 3. 機能一覧

### 3.1. リアルタイムスキャン機能（AR カメラ）

- カメラで物体をリアルタイム認識
- Google ML Kit / Vision API などで物体のカテゴリ分類（例：冷蔵庫、スマホ、ノート、ぬいぐるみ）
- 特定の物体カテゴリごとに GPT で異名＋裏設定を生成

---

### 3.2. 異名・裏設定生成機能（AI 連携）

- 物体カテゴリ名を GPT に渡し、以下の内容を生成：
  - 異名（例：「氷封の魔牢《フリージア・コア》」）
  - 属性（炎／闇／風など）
  - 簡単な設定テキスト（50〜100 文字）
  - 使用条件・封印解除文など（任意）
- GPT プロンプト例を元にランダム生成、またはカスタマイズモードも検討

---

### 3.3. スキャン演出（UI/UX）

- 魔法陣風 UI の演出（スキャン中に回転エフェクト）
- スキャン完了時に「封印解除」SE（効果音）、エフェクト発動
- 背景エフェクトは属性に応じて変化（例：炎 → 赤黒い背景）

---

### 3.4. コレクション（神器図鑑）

- スキャン履歴を保存
- 異名 / 属性 / 設定文 / スキャン日時 を記録
- 同じカテゴリを何度もスキャンすると設定が変化する仕様（再現性とバリエーション確保）

---

### 3.5. SNS シェア機能

- スキャン結果を画像化して保存・シェア可能（X, Instagram, TikTok 等）
- 「俺の冷蔵庫、実は魔具だった」などの投稿文自動生成オプション

---

### 3.6. 特殊イベント

- 特定の時間帯・日付で"超レア神器"が出現（例：午前 4:44、13日の金曜日、ハロウィン）
- 「次元歪曲モード」が突発的に発生し、演出が変化（BGM・異常設定など）

---

## 4. 技術スタック（推奨）

| 項目              | 技術候補                                       |
| ----------------- | ---------------------------------------------- |
| フロント          | Flutter（iOS / Android）                       |
| カメラ / 物体検出 | Google ML Kit / Vision API / TensorFlow Lite   |
| AI 生成           | OpenAI GPT API（Chat Completions）             |
| バックエンド      | Supabase or Firebase（図鑑保存、ユーザー管理） |
| シェア用画像生成  | Flutter Screenshot / html2canvas 相当技術      |
| エフェクト        | Lottie / Custom Animation / Flame（演出強化）  |

---

## 5. 画面構成（UI 構造）

1. **ホーム画面**

   - 「スキャン開始」ボタン
   - 図鑑、オプション、ヘルプなどに遷移

2. **スキャン画面（カメラ）**

   - スキャン対象物に魔法陣 UI
   - 検出後、自動で GPT 生成された結果表示

3. **神器結果画面**

   - 異名 / 属性 / 設定文 表示
   - 「コレクションに追加」ボタン
   - SNS シェアボタン

4. **図鑑画面**

   - カテゴリ別一覧
   - 日付・属性フィルター

5. **設定画面**
   - 言語、音声 ON/OFF、演出切り替えなど

---

## 6. ビジュアル / 演出イメージ（参考）

- ダークテーマ（黒＋金 or 黒＋紫）
- 魔法陣アニメーション
- 属性に応じた画面演出（雷なら稲妻、闇なら歪み）
- フォント：神秘的 / ルーン風 / 筆記体
- SE：封印解除音、召喚音など（無料 SE 素材を活用）

- 有料アドオン（広告削除 / 特殊エフェクト開放 / 前世神器召喚など）

## 9. 補足

- 初期生成プロンプトは AI に合わせて調整しながら運用
